# Keywords e Temas -- Estrategia SEO para FND

Analise de palavras-chave para fabiomorus.com. Foco em keywords FACEIS de ranquear para um site novo com baixa autoridade de dominio.

**Sistema de tracking:**
- Cada keyword tem um ID unico (coluna ID)
- Status: `disponivel` | `usado` | `reservado`
- Quando usar uma keyword, marcar como `usado` neste ficheiro E registar em `used.md`
- Para reservar uma keyword (em sessao ativa), marcar como `reservado`
- IDs do tipo `T1-EN-*` / `T1-PT-*` indicam que a mesma linha tem keyword EN e PT-BR separadas
- Verificar SEMPRE `used.md` antes de criar qualquer post

**Contexto competitivo:**
- Head terms ("functional neurological disorder", "transtorno conversivo") sao dominados por NHS, Wikipedia, Mayo Clinic, neurosymptoms.org
- O site NAO pode competir nessas queries agora
- Oportunidade: terapia especifica + FND, perguntas long-tail, mercado PT-BR (muito subatendido), localizacao (Jersey/Channel Islands)
- YMYL: Google exige E-E-A-T rigoroso. Cada post precisa demonstrar experiencia clinica, citar fontes medicas, ter author bio com credenciais

## Estrategia de Execucao Recomendada

### Mes 1-3: Quick Wins (Keywords Easy)
1. Comecar com as 15 keywords faceis do topo
2. Priorizar PT-BR (menos concorrencia, Fabio tem vantagem de native speaker)
3. Cada post: 1500-2500 palavras, schema Article, citacoes medicas, author bio com credenciais
4. Internal linking para pillar page futura

### Mes 4-6: Expandir Long-Tail
1. Cobrir as keywords Easy da Tier 3
2. Criar topic clusters (TNF + terapias, TNF + sintomas especificos)
3. Comecar pillar page em PT-BR (mais facil ranquear)

### Mes 7-12: Construir Autoridade
1. Mover para keywords Medium
2. Digital PR para backlinks
3. Pillar page em EN
4. Comecar a mirar em keywords Hard com conteudo definitivo

### Metricas de Sucesso
- 6 meses: Ranquear top 10 para 80% das keywords Easy
- 12 meses: 500+ sessoes organicas/mes non-branded
- 18 meses: Ranquear top 10 para keywords Medium
